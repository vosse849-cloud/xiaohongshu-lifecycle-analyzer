# 截图指南

截图请统一放在：

```text
docs/screenshots/
```

建议使用 mock 数据或完全脱敏数据截图，不要截图真实账号名称、真实后台数据、手机号、邮箱、Cookie、Session、API Key 或浏览器隐私信息。

## 推荐截图清单

文件名：

```text

```

- 当前观察时间输入框。
- 开始分析按钮。

## 截图尺寸建议

- 浏览器窗口宽度建议 1440px 或 1600px。
- 截图前切换到 `标准` 视觉模式。
- 不要使用过度缩放，保证文字可读。
- 文件统一使用 PNG。

## 上传前检查

截图进入 GitHub 前，请检查：

- 没有 API Key。
- 没有真实账号信息。
- 没有手机号、邮箱、地址、身份证。
- 没有真实后台数据。
- 没有浏览器个人信息。
